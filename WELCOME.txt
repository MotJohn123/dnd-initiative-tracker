================================================================================
    🚀 QUICK START COMMAND
================================================================================

1. Set up MongoDB Atlas (free):
   👉 https://www.mongodb.com/cloud/atlas/register

2. Edit .env.local with your MongoDB connection string

3. Run this command:
   npm run dev

4. Open your browser:
   http://localhost:3000

================================================================================
    🎮 FEATURES
================================================================================

✅ Public player view (NPCs shown as "?")
✅ DM admin panel with full control
✅ Create player groups
✅ Start & manage battles
✅ Set initiative values
✅ Add NPCs during combat
✅ Reveal/hide NPC names
✅ Lair actions (initiative 20)
✅ Turn tracking
✅ Mobile-friendly
✅ Auto-refreshing
✅ 100% FREE to host

================================================================================
    💻 TECHNOLOGY
================================================================================

Frontend:  Next.js 14 + React + TypeScript + Tailwind CSS
Backend:   Next.js API Routes + JWT Authentication
Database:  MongoDB Atlas (Free 500MB)
Hosting:   Vercel (Free)
